## Apps with this library

* [LapIcons](https://play.google.com/store/apps/details?id=com.lapism.lapicons)

## Api

 - minSdkVersion = 21
 - targetSdkVersion = 30
 - Java = 1.8
 - Kotlin = 1.8

Add the dependency to your gradle file:
```groovy
dependencies {
    implementation 'com.lapism:search2:1.0.0@aar'
}
```
